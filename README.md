Hangman Game 🎮

A simple Hangman Game built using Python. The player has to guess a hidden word one letter at a time before running out of attempts.

Features

- Random word selection
- Letter-by-letter guessing
- Input validation
- Limited attempts
- Win and lose conditions

Technologies Used

- Python 3
- Random Module

How to Run

1. Download or clone the project.
2. Open a terminal in the project folder.
3. Run:

python hangman.py

Game Rules

- Guess one letter at a time.
- You have 6 attempts.
- Correct guesses reveal letters in the word.
- Wrong guesses reduce remaining attempts.
- Guess the complete word before attempts run out.
